# Agente-inteligente-cazador-presa

Simulación de inteligencia artificial que modela la interacción entre dos agentes autónomos en una cuadrícula de 20×20. El Cazador usa el algoritmo A* para encontrar el camino óptimo hacia la Presa, mientras la Presa huye aplicando una estrategia greedy que maximiza su distancia al Cazador en cada paso.
Desarrollado en Python con Tkinter.
